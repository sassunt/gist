Initial release of a [conscript][cs] interface for pretty much the best pastebin ever.

### Features

- list gists
- view gists
- posts gists

### Install

     cs softprops/gist

### Get gisting

    gist auth login:password
    echo "Hey girl. I gist poems about you all the time." | gist --
